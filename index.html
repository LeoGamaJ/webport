Sobre Mim

Como desenvolvedor de sistemas integrados a Inteligência Artificial, combino expertise técnica com uma paixão por inovação. Minha experiência abrange desde o desenvolvimento web com Python (Flask/Django) até a integração de soluções avançadas de IA, utilizando tecnologias como GEMINI, OpenAI e Anthropic.

Trabalho na criação de aplicações web modernas, utilizando bancos de dados como Supabase e Firebase, e soluções em nuvem com Google Cloud Platform e Google Workspace. Meu foco é entregar soluções que não apenas atendam às necessidades atuais, mas também estejam preparadas para os desafios do futuro.
